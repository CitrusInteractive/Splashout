## About the Project

Splashout is a top-down paint-based game where players compete to cover the map in their colour. You move across a dotted grid-style arena, leaving paint trails, claiming space, and trying to dominate the match before time runs out or the arena becomes fully saturated.

The game is being built as a lightweight, experimental project focused on satisfying movement, smooth painting mechanics, and simple but competitive gameplay.

## Core Gameplay

Players move freely around a grid-based arena and paint the ground in real time. Every action leaves a visible trail, slowly turning the map into a layered mix of colours.

The main goal is to control as much space as possible. Movement speed, paint coverage, and positioning all matter, especially in tighter matches where space becomes limited.

## Features

- Real-time paint coverage system
- Grid-based dotted arena design
- Multiplayer support (planned / in development)
- Simple, fast movement mechanics
- Competitive territory control gameplay
- Colour-based visual feedback system

## Controls

- Move: WASD / Arrow Keys
- Paint: Automatic while moving (or hold LMB in some modes)
- Aim/Direction: Mouse (optional depending on mode)
- Menu: Esc

Controls are intentionally simple to keep the focus on movement and painting strategy rather than complex inputs.
